# Rastros de Jamaica

Este proyecto utiliza las flores como metáforas con el objetivo de analizar el impacto del tiempo de traslado en la vida cotidiana de los trabajadores del **Mercado de Jamaica**, uno de los principales centros florales de la **Ciudad de México**. El propósito es comprender cómo el cuerpo de quienes trabajan en el Mercado de Jamaica puede pensarse en paralelo al de las flores: ambos son organismos vivos que dependen del tiempo, el cuidado y las condiciones del entorno.
